Methodology

The inital data was a series of laz lidar return files which were decompressed into las format. Then, All the xyz point returns were binned into a DSM with 50cm horizontal resolution using the maximum return value if multiple points were contained in a grid cell. The resulting DSM was too large to perform solar irradiance calculations on a consumer workstation, therefore it was manually divided into 30 regions, which made irradiance calculation possible over the course of a month.
